# 小时工记账 App - 安装指南

## 快速安装 Node.js

由于自动安装遇到问题，请按以下步骤手动安装：

### 步骤 1：下载 Node.js
1. 打开浏览器访问：https://nodejs.org/
2. 点击绿色的 **LTS（长期支持版）** 按钮下载
3. 下载完成后，运行安装程序

### 步骤 2：安装 Node.js
1. 运行下载的安装程序
2. 点击 "Next" 继续
3. 接受许可协议
4. 选择安装路径（保持默认即可）
5. 点击 "Next" 完成安装

### 步骤 3：验证安装
安装完成后，**关闭所有终端窗口**，重新打开一个 PowerShell 窗口，输入：

```powershell
node -v
npm -v
```

应该显示类似这样的版本号：
- `v20.11.0`
- `10.2.4`

### 步骤 4：安装项目依赖
打开 PowerShell，进入项目目录：

```powershell
cd d:\Users\yang\Documents\ai\app
npm install
```

### 步骤 5：构建 Android App
依赖安装完成后，运行：

```powershell
npm run build:android
```

### 步骤 6：安装到手机
1. 确保手机已连接电脑，并开启 USB 调试
2. 打开 Android Studio（如果未安装，从 https://developer.android.com/studio 下载）
3. 打开 `d:\Users\yang\Documents\ai\app\android` 文件夹
4. 点击运行按钮，App 将安装到手机上

---

## 或者：直接使用 H5 版本

如果你只想在手机上使用，不需要安装原生 App：

1. 在电脑上运行：`npm run dev:h5`
2. 查看显示的本地地址（如 http://localhost:10086）
3. 在手机浏览器中访问该地址
4. 点击浏览器菜单，选择"添加到主屏幕"即可
